Monitors user activity. Shows a list of users and the number of posts, created for a specified period of time.

== Description ==

Plugin [Q3-Activity] allows you to view the activity of registered users. Displays a list of users with the number of records created for a specified period of time. By the click on the number of records created by an author you get a list of his posts on the page like this /wp-admin/edit.php

Later plugin functionality will be expanded.

== Installation ==

To set the plugin up, you need to:

1. Upload the folder `q3-activity` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

That's all!

== Frequently Asked Questions ==

= How to uninstall Q3-Activity? =
Simple deactivate it through the 'Plugins' menu in Wordpress
